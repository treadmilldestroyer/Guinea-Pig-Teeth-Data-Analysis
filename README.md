# Tooth Growth Data Analysis

This project analyzes a data set about tooth growth in guinea pigs and answers two questions, does vitamin C help in tooth growth in guinea pigs, and which method of vitamin C intake is better, ascorbic acid or orange juice?

## Instructions for Build and Use

[Software Demo](https://www.youtube.com/watch?v=NpQRcCiy9ng)

Steps to build and/or run the software:

1. There is nothing really to run in this software but clicking on the Run button will create the graphs in R.
2. Pressing the Knit button will produce a .html file that can be viewed in a new window or on a web browser.
